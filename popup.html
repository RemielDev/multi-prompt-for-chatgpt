Multi-Prompt for ChatGPT

Settings for the in-page buttons.

Opens 1–10 new chats each time.

Variation style

Chat 1 uses line 1, chat 2 uses line 2, and so on.

Where the buttons are

  • A grid icon next to ChatGPT's Send button.
  • A grid icon in each past message's hover toolbar.

Shortcuts

  • Ctrl/Cmd+Shift+1 — open this panel
  • Ctrl/Cmd+Shift+0 — close variation tabs
