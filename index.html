One prompt.
Every angle.

Send the same prompt to several new ChatGPT chats at once — each optionally with its own angle — and compare the answers side by side. Right inside ChatGPT, images and all.

View on GitHub How to install

Built for comparing answers

Stop asking the same question five times by hand. Fan it out and read the differences.

⊞

Fan out in one click

A grid button next to ChatGPT's Send opens your prompt in up to 10 new chats at once.

∠

Distinct angles

Give each chat its own angle — concise, detailed, critical, creative — or your own editable list.

↻

Re-run past prompts

Every previous message gets a button to re-send it — with its images — to fresh chats.

🖼

Media-safe

Re-attaches your images to every chat and refuses to submit one that's missing media.

⊟

Tidy tabs

Auto-groups the new chats and closes them all with one click or a keyboard shortcut.

🔒

Private

No account, no tracking, no servers. Prompts never touch disk and only ever go to ChatGPT.

Install in a minute

Until it's on the Web Store, load it unpacked:

Download or clone the GitHub repo.
Open chrome://extensions and turn on Developer mode.
Click Load unpacked and select the project folder.
Open chatgpt.com, type a prompt, and click the grid button next to Send.
